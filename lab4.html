Эксплуатация силового оборудования: электрических машин и комплектных устройств

🔬 Лабораторное занятие
⏱ 4 академических часа
📊 Сложность: высокая

Цель и задачи работы

Цель работы: Приобретение практических навыков по организации технической эксплуатации электрических машин (двигателей и генераторов) и комплектных распределительных устройств, включая диагностику технического состояния, анализ виброакустических характеристик, оценку состояния подшипников и изоляции, принятие решений по результатам испытаний.

Задачи работы

  1. Изучить нормативные требования к эксплуатации электрических машин и КРУ
  2. Освоить методику виброакустической диагностики электродвигателей
  3. Научиться анализировать результаты измерений сопротивления изоляции обмоток
  4. Приобрести навыки оценки состояния подшипниковых узлов
  5. Освоить методику планирования ремонтов электрических машин
  6. Научиться принимать обоснованные решения о допуске оборудования к эксплуатации

3. Оценка температуры подшипников

Нормативное обоснование: Согласно ГОСТ 183-74 (п. 7.3) и ПТЭЭП (п. 2.5.16):
  • Предельно допустимая температура подшипников качения — 100°С
  • Нормальная рабочая температура — не более 80°С
  • Превышение температуры над окружающей средой — не более 60°С
Анализ температуры подшипников:

Подшипник передний: T = 62°С < 80°С → Норма

Превышение над окружающей средой: ΔT = 62 - 25 = 37°С < 60°С → Норма

Подшипник задний: T = 78°С < 80°С, но близко к пределу

Превышение над окружающей средой: ΔT = 78 - 25 = 53°С < 60°С → Предупреждение

Вывод по п. 3: Температура заднего подшипника (78°С) приближается к предельному значению, что коррелирует с выявленным вибрационным дефектом. Повышенная температура подтверждает наличие дефекта подшипника.

4. Оценка состояния изоляции

4.1. Сопротивление изоляции
Нормативное обоснование: Согласно РД 34.45-51.300-97 (табл. 23, п. 23.1):
  • Для электродвигателей напряжением до 1000 В: Rиз ≥ 0,5 МОм
  • Измерение проводится мегаомметром на напряжение 500-1000 В

Измеренное значение: R60 = 125 МОм >> 0,5 МОм

Вывод: Сопротивление изоляции 125 МОм значительно превышает норму (в 250 раз). Состояние изоляции хорошее.
4.2. Приведение сопротивления изоляции к температуре 40°С
Нормативное обоснование: Для сравнения результатов измерений, выполненных при разных температурах, сопротивление изоляции приводится к стандартной температуре по формуле (РД 34.45-51.300-97, приложение 8):
Формула приведения к температуре 40°С:
R40 = Rt × KT
где: R40 — сопротивление при 40°С, МОм;
Rt — измеренное сопротивление при температуре t, МОм;
KT = 1,5(t-40)/10 — температурный коэффициент
Расчёт:

Температура измерения t = 45°С

KT = 1,5(45-40)/10 = 1,50,5 = 1,225

R40 = 125 × 1,225 = 153,1 МОм

4.3. Коэффициент абсорбции
Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 23.2):
  • Kабс ≥ 1,3 — изоляция сухая, в хорошем состоянии
  • Kабс = 1,0 — 1,3 — изоляция требует контроля
  • Kабс < 1,0 — изоляция увлажнена, эксплуатация недопустима

Измеренное значение: Kабс = 1,32

Вывод: Коэффициент абсорбции Kабс = 1,32 > 1,3 → Изоляция сухая, в хорошем состоянии

5. Оценка температурного режима двигателя

Нормативное обоснование: Согласно ГОСТ 183-74, для класса изоляции F:
  • Предельно допустимая температура обмотки — 155°С
  • Допустимое превышение температуры обмотки над охлаждающей средой — 105°С
  • При измерении по методу термометра на корпусе: Tкорп ≈ Tобм - (15...25)°С
Расчёт превышения температуры:

Температура корпуса: Tкорп = 72°С

Ориентировочная температура обмотки: Tобм ≈ 72 + 20 = 92°С

Превышение над охлаждающей средой: ΔT = 92 - 25 = 67°С

Допустимое превышение для класса F: 105°С

Запас: 105 - 67 = 38°С → Достаточный запас

Вывод: Температурный режим двигателя соответствует норме. Запас по превышению температуры составляет 38°С, что достаточно для работы при номинальной нагрузке.

Приложение Д. Характерные частоты дефектов подшипников качения

Обозначение Наименование Формула расчёта Дефект
fвр Оборотная частота fвр = n/60 Дисбаланс, несоосность
BPFO Частота перекатывания по наружному кольцу BPFO = z/2 × fвр × (1 - d/D × cos α) Дефект наружного кольца
BPFI Частота перекатывания по внутреннему кольцу BPFI = z/2 × fвр × (1 + d/D × cos α) Дефект внутреннего кольца
BSF Частота вращения тел качения BSF = D/(2d) × fвр × (1 - (d/D)² × cos²α) Дефект тел качения
FTF Частота вращения сепаратора FTF = fвр/2 × (1 - d/D × cos α) Дефект сепаратора
