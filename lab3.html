Цель и задачи работы

Цель работы: Приобретение практических навыков по организации технической эксплуатации трансформаторных подстанций (ТП) и распределительных пунктов (РП), включая планирование осмотров, диагностику оборудования, оценку состояния трансформаторного масла и принятие решений по результатам испытаний.

Задачи работы

  1. Изучить нормативные требования к эксплуатации ТП и РП
  2. Научиться анализировать результаты измерений и испытаний трансформаторов
  3. Освоить методику оценки качества трансформаторного масла
  4. Приобрести навыки составления технической документации
  5. Научиться принимать обоснованные решения о допуске оборудования к эксплуатации

  • I₀ = 2,6%

Решение

1. Приведение сопротивлений обмоток к температуре 20°С

Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 6.2) и ГОСТ 3484.1-88, измеренное сопротивление обмотки приводится к температуре 20°С для сравнения с паспортными данными.
Формула приведения сопротивления к температуре 20°С:
R₂₀ = Rt × (235 + 20) / (235 + t)
где: R₂₀ — сопротивление при 20°С, Ом;
Rt — измеренное сопротивление при температуре t, Ом;
t — температура обмотки при измерении (25°С);
235 — температурный коэффициент для медных обмоток (для алюминиевых — 228)
Расчёт для фазы AB:

R₂₀ = 4,28 × (235 + 20) / (235 + 25) = 4,28 × 255 / 260 = 4,28 × 0,9808 = 4,198 Ом

Расчёт для фазы BC:

R₂₀ = 4,32 × 255 / 260 = 4,32 × 0,9808 = 4,237 Ом

Расчёт для фазы CA:

R₂₀ = 4,25 × 255 / 260 = 4,25 × 0,9808 = 4,168 Ом

Результаты приведения:

Фаза Измерено при 25°С, Ом Приведено к 20°С, Ом Паспорт при 20°С, Ом
AB 4,28 4,198 4,12
BC 4,32 4,237 4,15
CA 4,25 4,168 4,10

2. Оценка соответствия сопротивлений обмоток нормам

2.1. Отклонение от паспортных данных
Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 6.2.1), отклонение измеренных значений сопротивления обмоток от паспортных данных не должно превышать ±2%.
Расчёт отклонений:

Фаза AB: δ = (4,198 - 4,12) / 4,12 × 100% = 0,078 / 4,12 × 100% = +1,89%

Фаза BC: δ = (4,237 - 4,15) / 4,15 × 100% = 0,087 / 4,15 × 100% = +2,10%

Фаза CA: δ = (4,168 - 4,10) / 4,10 × 100% = 0,068 / 4,10 × 100% = +1,66%

Вывод по п. 2.1: Отклонение фазы BC (+2,10%) незначительно превышает норму 2%. Согласно ПТЭЭП п. 2.1.17, при отклонении до 2,5% допускается эксплуатация с усиленным контролем. Рекомендуется повторное измерение для подтверждения результата.
2.2. Разброс между фазами
Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 6.2.1), отклонение сопротивления между фазами не должно превышать 2% от среднего значения.
Расчёт среднего значения и отклонений:

Среднее значение: Rср = (4,198 + 4,237 + 4,168) / 3 = 12,603 / 3 = 4,201 Ом

Фаза R, Ом Отклонение от среднего Оценка
AB 4,198 -0,07% ✓ Норма
BC 4,237 +0,86% ✓ Норма
CA 4,168 -0,79% ✓ Норма

Максимальный разброс: 0,86 - (-0,79) = 1,65% < 2%

Вывод по п. 2.2: Разброс между фазами (1,65%) не превышает допустимых 2%. Соответствует норме.

3. Анализ результатов измерения коэффициента трансформации

Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 6.1), отклонение измеренного коэффициента трансформации от паспортного значения не должно превышать:
  • ±0,5% для трансформаторов 110 кВ и выше
  • ±1,0% для трансформаторов до 35 кВ включительно

Паспортный коэффициент трансформации: Kтр = UВН / UНН = 10000 / 400 = 25,0

Фаза Kтр измер. Отклонение от паспорта Норма (±1,0%) Оценка
AB 25,12 +0,48% ±1,0% ✓ Соответствует
BC 25,08 +0,32% ±1,0% ✓ Соответствует
CA 25,15 +0,60% ±1,0% ✓ Соответствует
Вывод: Все отклонения коэффициента трансформации не превышают допустимых ±1,0%. Соответствует норме.

4. Оценка состояния изоляции

4.1. Сопротивление изоляции
Нормативное обоснование: Согласно РД 34.45-51.300-97 (табл. 6), для трансформаторов до 35 кВ сопротивление изоляции не нормируется, но должно быть не менее 10 МОм при мегаомметре 2500 В.
Параметр Измерено Норма Оценка
Rиз ВН-земля 2800 МОм ≥10 МОм ✓ Соответствует
Rиз НН-земля 1500 МОм ≥10 МОм ✓ Соответствует
4.2. Коэффициент абсорбции
Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 6.4), коэффициент абсорбции R₆₀/R₁₅ характеризует степень увлажнения изоляции:
  • Kабс ≥ 1,3 — изоляция сухая
  • Kабс = 1,0-1,3 — изоляция требует контроля
  • Kабс < 1,0 — изоляция увлажнена, эксплуатация недопустима

Измеренное значение: Kабс = 1,65 > 1,3

Вывод: Коэффициент абсорбции 1,65 > 1,3. Изоляция сухая, соответствует норме.
4.3. Тангенс угла диэлектрических потерь
Нормативное обоснование: Согласно РД 34.45-51.300-97 (табл. 10), для трансформаторов до 35 кВ tg δ изоляции обмоток не должен превышать 1,5% при температуре 20°С.

Измеренное значение: tg δ = 1,1% < 1,5%

Вывод: tg δ = 1,1% < 1,5%. Соответствует норме.

6. Оценка результатов опыта холостого хода

Нормативное обоснование: Согласно РД 34.45-51.300-97 (п. 6.6), потери холостого хода и ток холостого хода не должны превышать паспортных значений более чем на 5% для трансформаторов, находящихся в эксплуатации.
Расчёт отклонений:

Потери холостого хода:

δP₀ = (780 - 740) / 740 × 100% = 40 / 740 × 100% = +5,4%

Ток холостого хода:

δI₀ = (2,6 - 2,4) / 2,4 × 100% = 0,2 / 2,4 × 100% = +8,3%

Параметр Измерено Паспорт Отклонение Норма Оценка
P₀, Вт 780 740 +5,4% ≤5% ⚠️ Незначительное превышение
I₀, % 2,6 2,4 +8,3% ≤5% ⚠️ Превышение нормы
Анализ возможных причин (согласно СТО 56947007-29.180.01.015-2008):
  1. Ослабление прессовки магнитопровода
  2. Замыкания между листами электротехнической стали
  3. Повреждение изоляции стяжных шпилек
  4. Деформация магнитопровода
⚠️ Вывод по опыту ХХ: Требуется усиленный контроль параметров холостого хода при следующих испытаниях. Рекомендуется провести измерение сопротивления изоляции стяжных шпилек относительно магнитопровода.
